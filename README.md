📌 Sobre o Projeto

Este projeto é um **MVP (Minimum Viable Product)** de um aplicativo de lista de tarefas (To-Do List), desenvolvido com o objetivo de praticar conceitos de desenvolvimento mobile utilizando **Flutter**.

A aplicação permite que usuários organizem suas atividades do dia a dia de forma simples, rápida e eficiente.

O desenvolvimento foi realizado utilizando a plataforma **Antigravity**, focando em produtividade e prototipação ágil.

Funcionalidades

* ✅ Adicionar novas tarefas
* 📝 Editar tarefas existentes
* ❌ Remover tarefas
* ✔️ Marcar tarefas como concluídas
* 📋 Visualização simples e intuitiva da lista

## 🛠️ Tecnologias Utilizadas

* Flutter – Framework para desenvolvimento mobile multiplataforma
* Dart – Linguagem de programação
* Antigravity – Ambiente de desenvolvimento/prototipação

Layout e Usabilidade

O aplicativo foi projetado com foco em:

* Interface simples e limpa
* Experiência do usuário intuitiva
* Navegação rápida
* Facilidade no gerenciamento de tarefas

